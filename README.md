# fbb_ws63开发指南

## 介绍

  fbb_ws63代码仓为支持ws63和ws63e解决方案SDK，该SDK包从统一开发平台FBB（Family Big Box，统一开发框架，统一API）构建而来，在该平台上开发的应用很容易被移植到其他星闪解决方案上，有效降低开发者门槛，缩短开发周期，支持开发者快速开发星闪产品。

## 目录介绍

| 目录   | 介绍                                                 |
| ------ | ---------------------------------------------------- |
| docs   | 存放资料手册、IO复用关系表、硬件原理图、用户指南手册 |
